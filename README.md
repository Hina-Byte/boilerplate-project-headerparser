# Request Header Parser Microservice

Build a full stack JavaScript app that parses the request header information of the client. This project is part of the [FreeCodeCamp Backend Projects](https://www.freecodecamp.org/learn/back-end-development-and-apis/).

---

## API Endpoint

**Endpoint:** `/api/whoami`

**Method:** GET

### Response

Returns a JSON object with the following properties:

| Key        | Description                   |
|------------|-------------------------------|
| `ipaddress`| Your IP address               |
| `language` | Your preferred language       |
| `software` | Your system/user-agent info  |
